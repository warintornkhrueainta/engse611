Ordered and Unordered List Tags

Ordered List (OL)

Below is an ordered list, which numbers the list items:

  1. First item
  2. Second item
  3. Third item
  4. Fourth item
  5. Fifth item

Unordered List (UL)

Below is an unordered list, which uses bullets for list items:

  • Apple
  • Banana
  • Orange
  • Pineapple

Nested Lists

Below is a nested list where an ordered list is inside an unordered list:

  • Fruits
    1. Apple
    2. Banana
    3. Orange
  • Vegetables
    1. Carrot
    2. Broccoli
    3. Spinach
